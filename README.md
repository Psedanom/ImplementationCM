## Homework
- [E1 Syntax highlighter](https://github.com/Psedanom/ImplementationCM/tree/main/syntax_highlighter)
    - Syntax highlighter that takes a C++ source file as input and produces an HTML file with syntax highlighting. The highlighter doesn't use context free grammars, but instead relies on regular expressions to identify different token types in the C++ code. So multiline comments and other multiline things are not fully supported, but the highlighter can still identify them as a single token. The output HTML file includes CSS styles to color different token types differently, making the code easier to read and understand.
